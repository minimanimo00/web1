HTML

나중에 더 좋은 사이트로 찾아뵙겠습니다.

TAG

  • < / tag > : 영역 끝을 표시합니다.
  • < strong > : 굵은 글씨
  • < u > : 밑줄
  • < h1 >~< h6 > : 제목 별 글씨 크기(h1이 가장 큼)
  • < p > : 단락 바꾸기
  • < br > : 줄 바꾸기
  • < img width="100" src="이미지이름" >
      - img = 이미지를 넣을때 넣는 태그
      - width="이미지크기" = 이미지의 크기를설정(이름과 크기는 순서 바뀌어도됌)
      - src="이미지이름" = 이미지 문서 이름을 넣어 적용
  • < ul >< li >< /li >< /ul> = 부모와 자식같은 존재 ul은 바깥쪽으로 더 드러가서 분리시켜준다.
  • < li > = 목차같은 존재
  • < ol > = 목차의 순서를 자동으로 설정해준다.
  • < title > = 웹의 이름 설정
  • < a > = 링크
    < a href="링크복사"> ~ < /a > = (하이퍼 레퍼런스) 링크 붙이는 방법
    < a href="링크복사" target="_blank"> - < /a > = 링크를 누르면 탭이 생성된다.
    < a href="링크복사" target="_blank" title="html5 speicification" >
    = 링크가 무얼 가르키는지 밑에 뜬다.
  • < font color="색깔"> 원하는글 < /font> = 글자색 변경

    • 정의

  • Attribute = 속성
  • < ul,ol > = l=list의 약자, ordered list(ol), unordered list(ul)
  • < body > = 본문은 body태그로 묶기로 약속
  • < head > = 본문을 설명하는것을 head로 묶기로 약속
